计算机网络学习笔记（一）

发表于 2017-09-03 | 分类于 原理 ， 计算机网络
字数统计: 1,062 字

记录以备后用

计算机网络定义


  • 计算机网络就是互连的、自治的计算机集合。

    • 自治：无主从关系

    • 互连：通过通信链路互联互通



  • 距离远、数量大时，通过交换网络互联主机。

Internet定义

组成细节角度


  • 全球最大的互联网络

    • 由ISP(Internet Service Provider)网络互连的“网络之网络”



  • 数以百万计的互连的计算设备集合

    • 主机(hosts) == 端系统(end systems)

    • 运行各种网络应用



  • 通信链路

    • 光纤、铜缆、无线电、卫星……



  • 分组交换：转发分组(数据包)

    • 路由器(routers)

    • 交换机(switches)



服务提供角度


  • 为网络应用提供通信服务

    • Web, VolP, email, 电子商务, 社交网络



  • 为网络应用提供编程接口(API)

    • 支持应用程序“连接”Internet, 发送/接收数据

    • 提供类似于邮政系统的数据传输服务



网络协议

类比


  • 交通系统中的交通规则

  • 人类交谈作为信息交换的过程，同样包括交流时间、问题请教、人员引荐等

  • 网络通信

    • 通信主体：机器，而非人

    • 交换“数字化”或“电子化”信息

    • 所有通信过程必须遵循某些规则：协议



要素


  • 语法(Syntax)

    • 数据与控制信息的结构或格式

    • 信号电平



  • 语义(Semantics)

    • 需要发出何种控制信息

    • 完成何种动作以及做出何种响应

    • 差错控制



  • 时序(Timing)

    • 事件顺序

    • 速度匹配



Internet 协议标准


  • RFC：Request for Comments

  • IETF：互联网工程任务组(Internet Engineering Task Force)

计算机网络结构

  • 网络边缘
    • 主机（端系统）
    • 网络应用，如：Web，email
    • 模型：
      • 客户/服务器（client/server）应用模型
      • 对等（peer-peer，P2P）应用模型
  • 接入网络、物理介质
    • 有线或无线通信链路
      • 住宅（家庭）接入网络
      • 机构接入网络（学校、企业等）
      • 移动接入网络
    • 用户关心
      • 带宽（bps）
      • 共享/独占
    • 类型
      • 电缆网络 - 频分多路复用
      • 数字用户线路（DSL） - 电话线
      • Ethernet，端系统通常直接连接以太网交换机（switch）
      • 无线接入网络 - 通过基站或“接入点”
        • 无线局域网（LANs）
        • 广域无线接入 - 电信运营商（蜂窝网）
  • 网络核心（核心网络）
    • 互连的路由器（或分组转发设备）
    • 关键功能：
      • 路由（routing）
      • 转发（forwarding）

Internet 结构

  • 端系统通过接入ISP（access ISPs）连接到 Internet
    • 家庭、公司、大学 ISPs
  • 区域网络（regional networks）连接到接入ISP和运营商ISP
  • 每一个接入ISP连接到大型ISP
  • 大型ISP网络必须互连
  • 网络中心：少数互连的大型网络
    • “一级”（tier-1）商业ISPs（如：网通、电信、Sprint、AT&T），提供国家或国际范围的覆盖
    • 内容提供商网络（content provider network，如：Google），私有网络，连接其数据中心与Internet

数据交换

  • 电路交换
    • 典型：电话网络
    • 阶段
      • 建立连接
      • 通信
      • 释放资源
    • 特点：独占资源
    • 解决：多路复用
      • 特点
        • 将资源片分配给各路“呼叫”
        • 每路呼叫独占所分配资源
        • 资源片可能闲置（idle）
      • 方法
        • 频分多路复用（“frequency” FDM）
        • 时分多路复用（“time” TDM）
        • 波分多路复用（“wavelength” WDM）
        • 码分多路复用（“code” CDM）
          • 每个用户分配一个码片序列
          • 编码信号 = （原始数据）×（码片序列）
          • 各用户码片序列互相正交（orthogonal）

  • 报文交换

    • 报文：源（应用）发送信息整体

  • 分组交换

    • 分组：报文分拆出来的一系列相对较小的数据包
    • 分组交换需要报文的拆分与重组
    • 产生额外开销

  • 报文 vs 分组

    • 两者都是存储转发方式，区别：报文 - 完整报文，分组 - 较小的分组
    • 报文：M bits 链路带宽：R bps 分组长度：L bits 路由器数：n
      • 分组交换：T = M/R + nL/R
      • 报文交换：T = (n + 1)M/R
    • 分组交换允许更多用户同时使用网络，网络资源充分共享
    • 分组交换适用于突发数据传输网络，可能发生拥塞（congestion）：分组延迟和丢失，无电路级性能保障。
